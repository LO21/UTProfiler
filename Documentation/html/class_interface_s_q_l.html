Detailed Description

Classe permettant la communication avec la base de données.

La classe InterfaceSQL fait l'intermédiaire entre les différentes classes et notre base de données. Elle s'occuppe de s'y connecter, de charger les fichiers sql pour la remplir, d'exécuter les requêtes et de renvoyer les réponses sous forme de pointeur vers un objet.

Cette classe est un singleton, c'est à dire qu'un unique objet de cette classe peut exister à la fois et qu'un pointeur vers cet unique objet est disponible à tout moment à l'aide d'une méthode statique.
